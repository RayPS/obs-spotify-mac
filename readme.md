# OBS Spotify Mac
### An OBS script to display Spotify track info on macOS

_(Require OBS v28 and above. See [#2](https://github.com/RayPS/obs-spotify-mac/issues/2) if you are using older version)_

# Text Format
```
{n} = Name
{a} = Artist
```
# Artwork Image (Browser)
```
Width = 640
Height = 640
```
